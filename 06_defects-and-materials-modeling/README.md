Defects and Materials Modeling
==============================

This lesson applies LAMMPS, Atomsk, and OVITO to practical materials modeling
problems. Students will construct defects, relax structures, run simulations,
and interpret atomistic mechanisms from trajectories.

## Lesson Goals

1. Create vacancies, interstitials, surfaces, and grain boundaries.
2. Compute basic defect and surface energetics.
3. Simulate heating, deformation, diffusion, or relaxation workflows.
4. Use OVITO to identify structural changes and local atomic environments.

## Suggested Contents

* `student.ipynb`: Guided practical modeling exercises.
* `instructor-key.ipynb`: Completed workflow with validation notes.
* `atomsk/`: Structure generation commands.
* `lammps/`: Simulation inputs and potential files.
* `ovito/`: Visualization and analysis workflows.
* `data/`: Example structures, dumps, and reference outputs.
