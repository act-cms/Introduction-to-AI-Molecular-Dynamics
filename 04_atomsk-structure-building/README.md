Atomsk Structure Building
=========================

This lesson introduces atomistic model construction with Atomsk. Students will
create crystalline structures, convert between file formats, and prepare models
for LAMMPS simulations.

## Lesson Goals

1. Build FCC, BCC, HCP, and diamond structures with Atomsk.
2. Replicate, orient, and transform simulation cells.
3. Export structures to LAMMPS data format.
4. Introduce vacancies, substitutions, surfaces, and simple interfaces.

## Suggested Contents

* `student.ipynb`: Guided Atomsk commands and structure-building exercises.
* `instructor-key.ipynb`: Completed commands and expected outputs.
* `atomsk/`: Atomsk command scripts and generated structure files.
* `lammps/`: LAMMPS import examples using Atomsk-generated data.
* `figures/`: Structure snapshots and transformation diagrams.
