LAMMPS Fundamentals
===================

This lesson introduces practical molecular dynamics simulation with LAMMPS.
Students will learn the structure of a LAMMPS input script, run simple systems,
and connect simulation commands to physical modeling choices.

## Lesson Goals

1. Understand the main sections of a LAMMPS input script.
2. Create simple atomistic systems directly in LAMMPS.
3. Run minimization, NVE, NVT, and NPT simulations.
4. Read thermo output, dump files, and restart files.

## Suggested Contents

* `student.ipynb`: Guided walkthrough of LAMMPS concepts and scripts.
* `instructor-key.ipynb`: Completed lesson with annotated examples.
* `lammps/`: Input scripts, potential files, and run outputs.
* `data/`: Small example data files or reference outputs.
* `figures/`: Plots of energy, temperature, pressure, and structure.
