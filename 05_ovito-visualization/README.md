OVITO Visualization
===================

This lesson introduces visualization and trajectory analysis with OVITO.
Students will load LAMMPS output, apply common modifiers, and create
publication-quality images and animations.

## Lesson Goals

1. Load LAMMPS dump and data files into OVITO.
2. Color, select, slice, and filter atoms for clearer visualization.
3. Apply structural analysis modifiers such as CNA, coordination analysis, and
   atomic strain.
4. Export snapshots, plots, and trajectory movies.

## Suggested Contents

* `student.ipynb`: Guided visualization tasks and analysis questions.
* `instructor-key.ipynb`: Completed analysis workflow and expected results.
* `ovito/`: OVITO state files, scripts, or visualization notes.
* `data/`: Sample LAMMPS dump files for visualization.
* `figures/`: Rendered snapshots and animation frames.
