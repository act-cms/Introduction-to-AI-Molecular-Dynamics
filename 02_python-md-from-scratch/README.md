Python MD From Scratch
======================

This lesson introduces the core numerical pieces of molecular dynamics by
building a small MD code in Python. Students will implement particle positions,
velocities, pair forces, and time integration before comparing the behavior of
simple systems under different parameters.

## Lesson Goals

1. Implement basic MD data structures with NumPy.
2. Compute pairwise forces from simple potentials.
3. Use Velocity Verlet integration to propagate a trajectory.
4. Inspect energy conservation, stability, and time-step sensitivity.

## Suggested Contents

* `student.ipynb`: Guided coding activities for building the MD loop.
* `instructor-key.ipynb`: Completed version with explanations and checks.
* `data/`: Optional example inputs or saved trajectories.
* `figures/`: Generated plots and conceptual diagrams.
